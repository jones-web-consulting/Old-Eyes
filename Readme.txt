PRODUCT:
Old Eyes World of Warcraft Addon

VERSION:
2

DEVELOPED BY:
Kenneth R. Jones

DESCRIPTION:
This addon will modify the WoW interface to increase the size of the quest, gossip, and merchant frames, which will make reading the text much easier.

OPTIONS:
An in-game option is provided which allows the merchant scaling to be turned on or off.

INSTALLATION:
Copy the entire Old Eyes folder to your addon directory. By default, it should be located at: C:\Program Files (x86)\World of Warcraft\_retail_\Interface\AddOns

SUPPORT:
This product is not currently supported.
